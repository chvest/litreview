How it works
  1. Choose a stage, select the reviewers to assign, and set how many reviewers should cover each paper.
  2. Download the generated file(s). A single reviewer gets one CSV; multiple reviewers get a ZIP with one file each. The files include Title, Authors, Abstract, Year, DOI plus empty Decision and Notes columns.
  3. Send the files to each reviewer. They fill in the Decision column: 1 = include  ·  0.5 = uncertain  ·  0 = exclude.
  4. Import each completed file via Dashboard → Import → Import Reviews.
Tip: for rigorous reviews, set reviewers per paper to 2 or more. Papers are distributed with round-robin overlap so the workload stays balanced.
